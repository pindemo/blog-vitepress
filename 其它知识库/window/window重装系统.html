  • 网络驱动要下个，可能没有

  • U盘格式化 文件系统选择NTFS

  • 下载win11安装工具，选择Create Windows 11 Installation Media

  • 点击下载的工具，一路next，最后驱动器选择U盘

  • 进入电脑bios

重启，看到logo界面，按键都没用，就快速都点一遍。多试几次进入bios。

  • 保存后会重启进入安装系统页面

    选自定义

    全部删除，会清掉所有数据，然后选择想安装的硬盘，系统就会安装在指定硬盘

  • 安装完后会重启，需要在黑屏的时候数三下拔掉U盘，否则可能会重新在进行一次安装

  • 要联网，跳过
