安装TrollStore （不需要越狱 ​

字数: 0 字 时长: 0 分钟

准备 ​

选择版本 https://ios.cfw.guide/installing-trollstore/

只需要这两个就行了，其他没用。

  • 电脑安装 Sideloadly
  • 电脑下载ipa，TrollInstallerX

电脑安装 ​

  • 打开 Sideloadly ，iOS 设备插入电脑，选择受信任，window可以用转换头
  • 将 TrollInstallerX 文件拖放到 Sideloadly
  • 输入 Apple ID，其他不用管，直接点start
  • 然后输入密码
  • TrollInstallerX 将安装到 iOS 设备上

手机安装 ​

  • 信任应用程序 ，搜索VPN与设备管理就能看到
  • 开发者模式可能要开，建议开启再进行后面的。百度下怎么开。打开后手机会正常重启一次
  • 运行TrollInstallerX，有Install点就行了
  • 需要注意，有两个一定需要install
    • install ldid
    • install Persitence Helper (持久化的，选tips)
  • 看到安完上面两个就可以了
