安装tiktok ​

更新: 9/13/2024 字数: 0 字 时长: 0 分钟

无需拔卡，和国内一样使用。 首先下载tiktok，需要个国外的appleid。

设置 ​

更新: 9/13/2024 字数: 0 字 时长: 0 分钟

  • 安装tiktok
  • 设置里面的隐私，关闭tiktok追踪
  • 设置 > 通用 > 语言与地区，只需要把地区设为日本，其他的任何都不需要动
  • 小火箭开起来，别用跟国内有关的ip就行，比如香港

使用 ​

更新: 9/13/2024 字数: 0 字 时长: 0 分钟

  • 打开tiktok，看到推荐有视频就成功了
  • 注册建议appleid,其他可能有未成年检测，反正也没关系，有就重新注册个就行了
  • 多打开几次有定位提示，不要管
