Alist ​

字数: 0 字 时长: 0 分钟

下载 ​

下载后安装,安装完成后不用打开软件,若打开需要购买KEY,不用理会,关闭窗口就行

配置 ​

  • 进入安装目录,在安装目录的顶部栏输入cmd回车 打开终端

  • 命令行输入 alist.exe 回车 可查看一些基本信息

  • 命令行输入 alist.exe admin 回车 查看管理员密码

  • 命令行输入 alist.exe server 回车 启动项目

  • 浏览器打开http://localhost:5244/

  • 打开之后 账号输入 admin 密码是上面查出来的密码

  • 登录之后,底部有个管理按钮,点击进入配置

  • 首先就是将密码设置成我们自己的密码123456

  • 最重要的一点挂载我们本地硬盘,使其可以局域网访问

  • 保存后主页就能访问

开机自启 ​

  • 启动脚本.vbs
txt
Dim ws Set 
ws = Wscript.CreateObject("Wscript.Shell") 
ws.run "alist.exe server",vbhide 
Wscript.quit
  • 停止脚本.vbs
txt
Dim ws
Set ws = Wscript.CreateObject("Wscript.Shell")
ws.run "taskkill /f /im alist.exe",0
Wscript.quit

然后把这个快捷方式加入开机自启中，按住win+R，输入命令shell:startup
