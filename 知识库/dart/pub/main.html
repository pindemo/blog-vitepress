检查和运行 ​

shell
检查
flutter doctor
flutter --version

创建一个新项目
flutter create my_app

运行    修改代码后，保存，到终端输入r，就可以热重载项目
flutter run

检查已安装的设备
flutter devices

获取包
flutter pub get

升级flutter SDK和项目依赖包
flutter upgrade

flutter版本降级  downgrade后跟着指定版本
flutter downgrade 2.24.0
如果出现问题
flutter downgrade 2.24.0 There is no previously recorded version for channel "stable"
第一步、切换到稳定版
flutter channel stable
第二步、
flutter downgrade 2.24.0

查看当前项目可用的升级
flutter pub outdated

更新所有包
flutter pub upgrade

更新特定包
flutter pub upgrade <package_name>
如 flutter pub upgrade firebase_auth

将特定包更新为特定版本
flutter pub upgrade <package_name> <version>
如 flutter pub upgrade firebase_auth 1.0.0

运行(相当于下面的vscode配置)

构建和打包 ​

shell

flutter build ios

flutter build appbundle


创建包
https://dart.dev/guides/libraries/create-packages
flutter create --template=package 'package_name'
dart create -t package <PACKAGE_NAME>
如  dart create -t package ui 



插件
flutter create --template=plugin 'plugin_name'
flutter create -t plugin 'plugin_ui'
