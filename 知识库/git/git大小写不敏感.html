git大小写不敏感 ​

更新: 9/13/2024 字数: 0 字 时长: 0 分钟

使用命令 ​

sh
git config --get core.ignorecase
 
# 若为true值则代表当前是忽略的，此时需要将当前项目提交设置为false
 
git config core.ignorecase false

# 修改全局
git config --global core.ignorecase false

直接修改.git\config ​

txt
[core]
	ignorecase = false    //(将true改为false)
